# SheetNoProtection

SheetNoProtection is a simple C# WinForms application to remove protections from Excel Worksheets.

## Insatllation
Grab the latest release from [Releases](https://github.com/SheetNoProtection/releases).

## Building
Simply open the `.sln` file and build the project.
